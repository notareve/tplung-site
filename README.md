# TPlung Sensor Gateway Site

Статический сайт о функциональных возможностях TPlung и отдельная браузерная Live-панель.

Приложение TPlung в RuStore: https://console.rustore.ru/apps/1934742719

Контакты для вопросов и запросов ПО: raspi.collector@mail.ru

## Страницы

- `index.html` - главная страница с описанием TPlung, сенсоров, BLE advertising, USB диагностики и интеграций.
- `keenetic.html` - отдельная страница "Сценарий для роутеров" про использование TPlung как USB-донгла для Keenetic (MQTT, e-mail отчеты, этапы настройки).
- `live.html` - отдельная страница TPlung Live для работы с USB-стиком через Web Serial API.
- `firmware.html` - отдельная страница настройки TPlung через USB AT-команды; сейчас настройки устройства выполняются через BLE Settings characteristic.

Live-панель:

- подключается к USB-стикy через Web Serial API;
- работает поверх USB Full-Speed 12 Мбит/с в режиме CH340;
- управляет TPlung через AT-команды (`atXXXX`);
- парсит ADV пакеты (Thermo/Humi/Энергомера CE308);
- показывает данные карточками.
- хранит локальные имена устройств и каналов в браузере;
- проверяет Live-значения по пользовательским Min/Max порогам, контролирует отсутствие устройства и подсвечивает активные уведомления;
- хранит локальный архив трендов, экспортирует весь архив или видимый период в CSV.

## Как запустить

1. Откройте папку проекта `Project/web-serial-thermo`.
2. На Windows можно просто запустить `start-localhost.bat`.
3. Или запустите любой локальный HTTP-сервер в этой папке.
4. Откройте `http://localhost:PORT` в Chrome/Edge.
5. Для Web Serial панели перейдите на `http://localhost:PORT/live.html`.
6. Для страницы настройки по USB перейдите на `http://localhost:PORT/firmware.html`.

Примеры локального сервера:

- `npx serve .`
- `python -m http.server 8080`

## Важно

- Web Serial API работает в Chromium-браузерах.
- Нужен безопасный контекст: `https://` или `http://localhost`.
- Прямой запуск файла `index.html` через `file://` не подойдет.
- Выбор serial-порта разрешается пользователем через кнопку "Подключить порт".
- Для публикации на `kompius.ru` размещайте файлы под одним HTTPS-доменом, например `/tplung/`, чтобы `localStorage` и разрешения Web Serial не дробились между `www` и non-`www`.

## Использование с Keenetic

- TPlung можно использовать как USB-донгл для Wi-Fi роутеров Keenetic.
- На роутере настраивается публикация данных в MQTT брокер (удобно для Home Assistant и внешней аналитики).
- Поддерживаются ежедневные отчеты на e-mail по cron-расписанию (текст/таблицы и графики в зависимости от сценария).
- Дополнительно доступны локальный архив измерений на USB-накопителе и автозапуск сервисов после перезагрузки роутера.
